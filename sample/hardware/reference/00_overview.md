# CHINJU Hardware Reference Designs - Overview

## 目的

本ディレクトリは、CHINJUハードウェア層の**リファレンス設計**を提供する。

- **硬い要件**は `REQUIREMENTS.md` で定義済み
- 本ディレクトリは**実装例**を示す（必須ではない）
- パートナーは独自設計も可能（要件を満たせば）

---

## 設計原則（再掲）

### 生存原理の物理層適用

```
┌─────────────────────────────────────────────────────────┐
│  多様性（N_eff）                                         │
│  ────────────────────────────────────────────────────── │
│  • 各機能に複数ベンダー選択肢                            │
│  • 1社依存禁止                                          │
│  • 代替調達経路の確保                                    │
└─────────────────────────────────────────────────────────┘
                    ↕ 独立
┌─────────────────────────────────────────────────────────┐
│  独立性（DIP）                                           │
│  ────────────────────────────────────────────────────── │
│  • トレイト（抽象）への依存                              │
│  • 具体実装の隠蔽                                        │
│  • 設定による切り替え                                    │
└─────────────────────────────────────────────────────────┘
                    ↕ 交換
┌─────────────────────────────────────────────────────────┐
│  交換可能性                                              │
│  ────────────────────────────────────────────────────── │
│  • コード変更なしで実装交換                              │
│  • 認証済み実装間の互換性                                │
│  • フォールバック構成                                    │
└─────────────────────────────────────────────────────────┘
```

## アーキテクチャパターン

### Pattern A: オンプレミス統合型

```
┌──────────────────────────────────────────────────────┐
│  Host Server                                         │
│  ┌────────────────────────────────────────────────┐  │
│  │  CHINJU Software Layer                         │  │
│  │  (Rust)                                        │  │
│  └─────────────┬──────────────────────────────────┘  │
│                │ FFI / PKCS#11                       │
│  ┌─────────────┴──────────────────────────────────┐  │
│  │  Hardware Abstraction Layer                    │  │
│  └─────────────┬──────────────────────────────────┘  │
│                │                                     │
│  ┌─────────────┼──────────────────────────────────┐  │
│  │             ▼                                  │  │
│  │  ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐         │  │
│  │  │ HSM  │ │ TEE  │ │ QRNG │ │ OTP  │         │  │
│  │  └──────┘ └──────┘ └──────┘ └──────┘         │  │
│  │  PCIe/USB  SGX      PCIe    MCU内蔵           │  │
│  └────────────────────────────────────────────────┘  │
└──────────────────────────────────────────────────────┘
```

**用途**: データセンター、エンタープライズ
**レベル**: Level 2-4

### Pattern B: クラウドHSM + ローカルTEE

```
┌──────────────────────────────────────────────────────┐
│  Cloud (AWS/Azure/GCP)                               │
│  ┌──────────────────────────────────────────────┐    │
│  │  CloudHSM / Managed HSM                      │    │
│  └──────────────────────┬───────────────────────┘    │
└─────────────────────────│────────────────────────────┘
                          │ TLS 1.3
┌─────────────────────────│────────────────────────────┐
│  Local Server           │                            │
│  ┌──────────────────────┴───────────────────────┐    │
│  │  CHINJU Software Layer                       │    │
│  │  (Rust)                                      │    │
│  └──────────────────────┬───────────────────────┘    │
│                         │                            │
│  ┌──────────────────────┼───────────────────────┐    │
│  │  ┌──────┐ ┌──────┐   │                       │    │
│  │  │ TEE  │ │ OTP  │   │                       │    │
│  │  └──────┘ └──────┘   │                       │    │
│  └──────────────────────────────────────────────┘    │
└──────────────────────────────────────────────────────┘
```

**用途**: ハイブリッドクラウド、スタートアップ
**レベル**: Level 2-3

### Pattern C: エッジデバイス

```
┌──────────────────────────────────────────────────────┐
│  Edge Device (ARM)                                   │
│  ┌────────────────────────────────────────────────┐  │
│  │  MCU: STM32L5 / NXP LPC55S / Nordic nRF5340   │  │
│  │  ┌──────────────────────────────────────────┐  │  │
│  │  │  TrustZone Secure World                  │  │  │
│  │  │  ┌──────────────────────────────────┐    │  │  │
│  │  │  │  CHINJU Core (Rust, no_std)      │    │  │  │
│  │  │  └──────────────────────────────────┘    │  │  │
│  │  │  ┌────────┐ ┌────────┐ ┌────────┐       │  │  │
│  │  │  │ SE     │ │ OTP    │ │ TRNG   │       │  │  │
│  │  │  │ATECC608│ │ eFuse  │ │ 内蔵   │       │  │  │
│  │  │  └────────┘ └────────┘ └────────┘       │  │  │
│  │  └──────────────────────────────────────────┘  │  │
│  └────────────────────────────────────────────────┘  │
└──────────────────────────────────────────────────────┘
```

**用途**: IoT、組み込み、ロボティクス
**レベル**: Level 1-2

### Pattern D: 完全エアギャップ

```
┌─────────────────────────────────────────────────────┐
│  Secure Zone (SCIF)                                 │
│  ┌───────────────────────────────────────────────┐  │
│  │  Air-Gapped Server                            │  │
│  │  ┌─────────────────────────────────────────┐  │  │
│  │  │  CHINJU + HSM + TEE + QRNG              │  │  │
│  │  └─────────────────────────────────────────┘  │  │
│  └───────────────────────┬───────────────────────┘  │
│                          │ Data Diode (Unidirectional)
│  ┌───────────────────────▼───────────────────────┐  │
│  │  Outbound Only                                │  │
│  └───────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────┘
          │ 物理的に一方向
          ▼
┌─────────────────────────────────────────────────────┐
│  External Network                                   │
└─────────────────────────────────────────────────────┘
```

**用途**: 国家機密、軍事、重要インフラ
**レベル**: Level 4
